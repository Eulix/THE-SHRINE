# The sum of the first n integers
$$
\sum_{k=1}^{n}k = 1+2+3+....+ n = \frac{n(n+1)}{2}
$$


# The sum of the squares of the first n integers
$$
\begin{align}
\sum_{k=1}^{n}k^2 = 1^2+2^2+3^2+....+ n^2  \\
= \frac{n(n+1)(2n+1)}{6} \\
= \frac{1}{3}n^3 + \frac{1}{2}n^2 + \frac{1}{6}n
\end{align}
$$
